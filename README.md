Учебный проект. Попробуйте здесь: https://elsiesmile.github.io/bem-registration-form/

Главные задачи в исполнении данного проекта:

Научиться пользоваться методологией BEM, закрепить свои знания на практике; 
Эта технология позволяет использовать блоки повторно, понимать свой код и других коллег, делает проект понятным и ускоряет процесс разработки;

В результате проект реализован, навыки получены.
